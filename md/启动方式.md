- 启动方式（PowerShell）:
- 建议虚拟环境后运行:
- python -m pip install -r requirements.txt
- python -m streamlit run app/streamlit_app.py
- 或先设 Key 再运行:
- $env:GEMINI_API_KEY="你的APIKey"
- python -m streamlit run app/streamlit_app.py
