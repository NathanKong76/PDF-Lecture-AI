# Pandoc PDF 生成失败问题分析

## 错误信息
"Pandoc PDF generation failed or unavailable"

## 可能导致问题的原因列表

### 1. 工具可用性问题
- **Pandoc 未安装或不在 PATH 中**
- **XeLaTeX 未安装或不在 PATH 中**
- **缓存状态错误**（工具已安装但缓存显示不可用）
- **工具路径错误**（路径包含空格或特殊字符）

### 2. 文件路径和编码问题
- **临时文件路径包含特殊字符**（空格、中文、特殊符号）
- **字体文件路径问题**（路径不存在、权限不足、编码问题）
- **文件编码问题**（UTF-8 vs Windows-1252）
- **路径长度限制**（Windows 260字符限制）

### 3. LaTeX 编译错误
- **LaTeX 模板语法错误**
- **字体加载失败**
- **包依赖缺失**（xeCJK、multicol 等）
- **特殊字符未转义**（&, %, $, #, ^, _, {, } 等）
- **表格/代码块格式问题**
- **数学公式语法错误**

### 4. Pandoc 转换错误
- **Markdown 内容包含未转义的 LaTeX 特殊字符**
- **Pandoc 版本不兼容**
- **模板文件路径问题**
- **输入内容编码问题**

### 5. 超时问题
- **内容过大导致超时**
- **LaTeX 编译时间过长**
- **超时设置过短**

### 6. 内存和资源问题
- **内存不足**
- **临时文件过多**
- **进程资源限制**

### 7. 错误处理不完善
- **错误信息未正确传递**
- **异常被吞没**
- **日志信息不足**

### 8. 边界情况
- **空内容**
- **极小的尺寸**
- **极大的内容**
- **特殊文件名**

### 9. 平台特定问题
- **Windows 路径分隔符**
- **文件权限问题**
- **Shell 命令执行问题**

### 10. 并发问题
- **多进程同时访问缓存**
- **临时文件冲突**
